    Features

    Calculator v0.1β

      Used MikeMcl's decimal.js library:
    • https://github.com/MikeMcl/decimal.js

      • Currently supported operators:
    • Basic: 
      + : Plus (+)
- : Minus (−) 
* : Times (×) 
/ : Divide (÷) 
% : Module
    • Extanded: 
      ** : Power
// : Integer Divide
    • Bitwise (integers only): 
      ~ : Bitwise NOT
& : Bitwise AND
| : Bitwise OR
^ : Bitwise XOR
    • Logical (true = 1, false = 0): 
      ! : Logical NOT
&& : Logical  AND
|| : Logical  OR
    • Relational (true = 1, false = 0): 
      == : Equal
!= : Not equal
> : Greater than
< : Less than
>= : Greater than or Equal to
<= : Less than or Equal to
      • Operator Orders:
    • Use ( and ) to promote orders. 
      1	**
2	* / % //
3	+ -
4	~ & | ^
5	! && ||
6	== != > < >= <=
